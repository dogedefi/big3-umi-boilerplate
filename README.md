# umi project

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

## Create Component


Supported by [`big3-new-component`](https://www.npmjs.com/package/big3-new-component), Usage

```bash
yarn generate Example -d src/pages
```
